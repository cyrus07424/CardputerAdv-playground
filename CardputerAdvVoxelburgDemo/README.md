# CardputerAdvVoxelburgDemo

`Attack on Voxelburg` 風デモです。

## このデモでできること

- ボクセル風の起伏地形の上を低空飛行
- 敵防衛ポッドをすべて撃破してクリア
- 敵のチャージ攻撃を横移動で回避
- ゲームオーバーなし・射撃クールダウンなしで飛び続けられる `ENDLESS` モード
- 難易度 `EASY / NORMAL / HARD` 切替
- Cardputer-ADV 本体キーボードだけで操作

## 必要なライブラリ

- M5Cardputer

## Arduino IDE 設定

- 開くファイル: `CardputerAdvVoxelburgDemo.ino`
- ボード: `m5stack:esp32:m5stack_cardputer`

## 操作

1. `A` で前進
2. `S` で射撃
3. `Q` / `E` で左右ストレーフ
4. `,` / `/` または `←` / `→` で左右旋回
5. `;` / `.` または `↑` / `↓` で視点上下
6. タイトル画面で `Z` / `X` を押すと難易度変更
7. タイトル画面で `C` を押すと `MISSION / ENDLESS` 切替
8. `Enter` で開始、リザルト画面からタイトルへ戻る
9. `R` でミッション再生成
10. `H` でヘルプ表示切替

## 補足

- 地形・色・敵配置は手続き生成です。
- 高度は地形に追従する固定高度飛行として再構成しています。
- スコアは難易度ボーナス、残り耐久、クリア時間を元に計算します。
- `ENDLESS` では被弾しても撃墜されず、射撃クールダウンもなく、現在の敵を全滅させると次ウェーブが自動生成されます。
