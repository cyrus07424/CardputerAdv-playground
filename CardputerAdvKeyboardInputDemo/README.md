# CardputerAdvKeyboardInputDemo

Cardputer-ADV 本体キーボードから文字入力する、最小構成の Arduino デモです。

## 必要なライブラリ

- M5Cardputer

## Arduino IDE 設定

- 開くファイル: `CardputerAdvKeyboardInputDemo.ino`
- ボード: `m5stack:esp32:m5stack_cardputer`

## 使い方

1. 書き込み後、Cardputer-ADV 本体キーボードで文字入力する
2. Enter で現在の行を確定して履歴へ移動する
3. BtnA で入力内容と履歴をクリアする
