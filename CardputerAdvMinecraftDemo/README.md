# CardputerAdvMinecraftDemo

Minecraftの雰囲気を Cardputer-ADV 向けに落とし込んだ、Arduino IDE 用の軽量 Minecraft 風デモです。

Cardputer-ADV で成立する範囲に絞って再構成しています。

## このデモでできること

- 一人称視点で小さなボクセル世界を移動
- より広いボクセル世界を移動
- 草原 / 森 / 砂漠 / 赤砂漠 / 雪原 / 岩場 / 湿地を含む地形生成
- 木 / 針葉樹 / サボテン / 岩塊のような簡易構造物生成
- 草や花のような軽量な植物デコレーション表示（草・花は高解像度 cutout、花は複数バリエーション）
- 狙っているブロックの破壊 / 設置
- Cardputer-ADV 本体キーボードだけで操作

## 省略しているもの

- 無限チャンク生成
- セーブ / ロード
- 音 / GUI メニュー / 昼夜サイクル

## 必要なライブラリ

- M5Cardputer

## Arduino IDE 設定

- 開くファイル: `CardputerAdvMinecraftDemo.ino`
- ボード: `m5stack:esp32:m5stack_cardputer`

## 操作

1. `WASD` で移動
2. `J` / `L` で左右旋回、`I` / `K` で視点上下
3. `G` でジャンプ
4. `F` で飛行モード切替、飛行中は `Q` / `E` で下降 / 上昇
5. `Z` でブロック破壊、`X` で設置、`C` で設置ブロック切替
6. `R` でワールド再生成
7. `H` でヘルプ表示切替
8. `U` でヘッダー表示切替
9. `B` でフッター表示切替
10. `Fn` で設定ポップアップ表示、表示中は **カーソルキー**（`↑↓←→`。Cardputer-ADV では `; . , /`）で **視野角 (70-100 度)**、**横移動速度 (0.5x-5.0x)**、**昼 / 夜** を変更し、**SAVE / LOAD** で SD カードへワールド保存 / 読込

## 補足

- 描画は **複数のレンダ解像度** を切り替えられ、各ボクセル面はスケッチ内蔵の簡易 8x8 テクスチャで表現しています。
- ワールドサイズを広げ、追加ブロックとして **Cobble / Planks / Snow / Clay / RedSand** も扱えるようにしています。
- SD カードの `/mcadv/world.bin` に現在のワールド状態を書き出し / 読み戻します。
- SD 初期化は起動直後ではなく、**Fn の設定ポップアップを最初に開いた時** に行います。
- 静止中は **240x135**、移動中や視点変更中は **120x68 / 80x45 / 60x34** に自動で落として速度を優先します。
- ブロックの破壊 / 設置 / 選択ブロック切替を行った直後も、再描画を速くするため **1 フレームだけ低解像度** で描画します。
- 移動中や視点変更中に **15 FPS を下回る** と、移動用解像度はさらに下の段へ自動で下がります。
- ネイティブ解像度の計算中に移動や視点変更の入力が入った場合は、現在の高解像度フレーム計算を打ち切って次のフレームへ移ります。
- 描画中に短く押したキー入力も次フレームへラッチするため、レンダリング中の短押しが取りこぼされにくくなっています。
- カメラの向き計算には `Pescado-Engine-Embedded-ESP32S3` 由来の **Vector / Matrix ベースの 3D 数学コア** を移植して使っています。
- 元実装のブロック系ゲームプレイを Cardputer-ADV 上で触れることを優先し、機能はかなり絞っています。
