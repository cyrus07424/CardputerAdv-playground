# AnalogClockDemo

クロノグラフアナログ時計

## 必要なライブラリ

- M5Cardputer

## Arduino IDE 設定

- 開くファイル: `AnalogClockDemo.ino`
- ボード: `m5stack:esp32:m5stack_cardputer`

## 操作

1. `A`: ストップウォッチ開始 / 再開
2. `S`: ストップウォッチ停止
3. `D`: スプリット表示の固定 / 解除
4. `R`: 停止中のストップウォッチをリセット
5. `Enter` または `BtnA`: ヘルプ表示の切替

## メモ

- メインダイヤルには**時針 / 分針**と、クロノグラフの**センター秒針**を重ねて表示します
- 上サブダイヤルは 1/10 秒・1/100 秒、左サブダイヤルは時計の秒、下サブダイヤルは 0〜5 分の経過を表示します
- ボード側で有効な時刻が取得できる場合はそれを使い、未設定時は**ビルド時刻**を初期値として進めます
