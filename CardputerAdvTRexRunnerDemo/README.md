# CardputerAdvTRexRunnerDemo

Cardputer-ADV 向けの `T-Rex Runner` 風デモです。  

## 必要なライブラリ

- M5Cardputer

## Arduino IDE 設定

- 開くファイル: `CardputerAdvTRexRunnerDemo.ino`
- ボード: `m5stack:esp32:m5stack_cardputer`

## 現在の内容

- T-Rex のジャンプ / しゃがみ
- 雲、地面、サボテン、鳥を含む endless runner
- 速度上昇とスコア / ハイスコア表示
- ゲームオーバーとリスタート

## 操作

1. `A` または `;` でジャンプ
2. `.` でしゃがみ / 落下加速
3. `Enter` または `BtnA` でリスタート
