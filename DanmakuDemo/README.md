# DanmakuDemo

弾幕シューティングデモです。

## 必要なライブラリ

- M5Cardputer

## Arduino IDE 設定

- 開くファイル: `DanmakuDemo.ino`
- ボード: `m5stack:esp32:m5stack_cardputer`

## 操作

1. `, / ; .` または 矢印キー: 自機移動
2. `A`: ショット
3. `S`: 低速移動 / hitbox 表示
4. `D`: ボム
5. `Enter` または `BtnA`: タイトル / クリア / ゲームオーバー画面から開始・再開

## 内容

- 左プレイフィールド + 右 HUD の PC 版風レイアウト
- 通常移動と低速移動、パワー段階で変化する自機ショット
- 敵弾のグレイズ加点
- 通常敵ウェーブからボス戦へ進む 1 ステージ構成
- ボムによる弾消しとボス / 敵への緊急ダメージ
